Fresh-Mart

Fresh-Mart is a modern full-stack e-commerce grocery platform designed to provide a smooth and responsive online shopping experience for fresh products and daily essentials.

## Live Demo

🔗 https://fresh-mart-qt0suovh9-ayush-projects1.vercel.app

---

# Features

*  Browse fresh grocery products
*  Product search and filtering
*  Fully responsive UI
*  Fast and modern frontend using Vite
*  Organized product categories
*  Shopping cart functionality
*  Clean and user-friendly interface

---

#  Tech Stack

## Frontend

* React.js
* Vite
* CSS / TailwindCSS

## Backend

* Node.js
* Express.js

## Database

* MongoDB

# Installation & Setup

## Clone Repository

```bash
git clone https://github.com/Ayush26-03/Fresh-Mart.git
```

---

## 2️ Navigate to Project

```bash
cd Fresh-Mart
```

---

#  Frontend Setup

```bash
cd client
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```

---

#  Backend Setup

```bash
cd backend
npm install
npm start
```
